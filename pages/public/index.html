lopecode

A self-serializing notebook format that publishes to the atmosphere.

This origin is the public surface for the lopecode network: profile pages, OAuth, the per-DID web proxy, an indexer, and a feed generator. Bundles themselves live on each author’s PDS as com.lopecode.bundle records.

Where to start

  • github.com/tomlarkworthy/lopecode-dev — design notes, specs, and the agentic dev harness.
  • specs/atproto.md — the v1 atproto integration plan this domain implements.
  • atproto.html — in-notebook publisher and reader, runs entirely client-side.
